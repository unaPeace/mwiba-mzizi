My Journey with JavaScript through Games.

The thrill of creating interactive worlds with challenges to overcome, fueled my desire to learn the JavaScript.

With each game I built, I tackled new concepts in JavaScript, from basic mechanics to animation and user input. It was a fun and immersive way to grasp the fundamentals and apply them in a creative space.

This journey not only equipped me with the technical skills of JavaScript, but also instilled a problem-solving approach and a passion for bringing ideas to life through code.

This program is a 3D hack and slash game demo built with JavaScript and Three.js. It creates a virtual world with a player character, enemies (NPCs), plants, and visual effects.

This program starts by building visuals effects (lighting, fog, sky), then a player character and enemies. It spawns enemies and plants, and implements a third-person camera following the player.

02. Banana Boom

This is a two-player, turn-based artillery game where you aim bombs at your opponent's gorilla in a destructible cityscape. Players take turns throwing bombs, trying to hit the gorilla directly or with the blast.

The game offers a simulation mode for single-player against a computer AI that calculates the best throw for increased difficulty. Wind speed is also a factor, making aiming more challenging.

This Javascript program creates a web-based maze game. Players navigate balls through a maze using a joystick control. The maze is built with HTML and CSS, and balls are affected by gravity and friction.

Players can choose easy mode or hard mode, with black holes added for difficulty in hard mode. The game provides feedback on progress and completion.

Points are awarded for successful crossings, with double points for perfect timing. The game ends if the character falls.

05. Space Invaders

This program is the classic Space Invaders game using HTML5 canvas and JavaScript. Players control a spaceship at the bottom of the screen shooting down alien invaders that move across the screen.

Points are awarded for each destroyed alien. The game ends if an alien reaches the bottom.
